What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of an HTML or XML document as a tree of objects, where each node is an element, attribute, or text within the document. The DOM allows programming languages like JavaScript to interact with and manipulate the content, structure, and style of a webpage dynamically.

How do you select an element from the DOM?

To select an element from the DOM using JavaScript, one can use several built-in methods provided by the document object. Here are the most common methods:getElementById, getElementsByClassName, getElementsByTagName, querySelector, and querySelectorAll.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique where a single event listener is attached to a parent element to manage events for multiple child elements. It is useful because it improves performance by reducing the number of event listeners and allows dynamic elements (added after page load) to be handled efficiently.

How do you manipulate an element's attributes and styles using the DOM?

You can manipulate an element's attributes using methods like setAttribute(), getAttribute(), and removeAttribute(). To manipulate styles, you can directly modify the style property of the element or use classes via classList.
